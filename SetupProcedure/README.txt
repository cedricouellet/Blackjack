INSTALLATION PROCEDURE
=========================================================
1. unzip the archive
2. make sure the 'CAO.Blackjack.Setup.msi' file is in the same folder as 'setup.exe'.
3. open 'setup.exe'
4. Proceed with the dialog to install the application.

WARNING
==========================================================
Do not start the 'CAO.Blackjack.Setup.msi' file manually, 
as it is only a dependency to the 'setup.exe' file.
